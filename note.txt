Step to run project:

1) Move Knowledge Bank Project to htdocs folder(Inside XAMPP Server)

2) Open XAMPP Control Panel and start Apache & MySQL Server

3) Open any browser and type localhost

4) Open PHPMyAdmin (or type http://localhost/phpmyadmin/)

5) Create 'knowledgebank' database and then click on Import option

6) Choose the knowledgebankDB (SQL)file inside the knowledgebank folder

7) Then click on Go button at bottom (Now DB successfully created)

8) Now again type http://localhost/knowledgebank/ to view your project

9) Congrats, now your project successfully run...
